5 Exploratory Data Analysis

Several R packages provide functions with identical names, such as describe() in the Hmisc package and describe() in the psych package. The first function performs descriptive analysis by providing robust estimators and shape metrics, such as the interquartile range (IQR), trimmed mean, median absolute deviation (MAD), skewness, and kurtosis. In contrast, the second function provides a distribution profile using percentiles, Gini’s mean difference (Gdm), and the pseudomedian (pMedian), while also facilitating data screening with diagnostics for unique and missing values. The epiDisplay package also provides functions inspired by Stata, such as summ(). Additional valuable resources include the janitor package for data cleaning, as well as initiatives like the R Epidemics Consortium (RECON), Epiverse-TRACE, and the R4EPI project, which provide specialized tools for epidemiological research and analysis.

6.4 Table 2. Multivariable analysis

Incluir el departamento como una variable predictora (efecto fijo) es la mejor opción porque, dado el número reducido de unidades de nivel superior (\(K=4\) clústeres/departamentos), un modelo de efectos mixtos no permite estimar de forma fiable la varianza inter-departamental (\(\tau^2\)). Asimismo, los estimadores de varianza robust como vcovCL (basados en la matriz de covarianza sándwich asintótica tipo HC0 a HC3) dependen de propiedades que requieren de 30 a 50 clusters clústeres para evitar la subestimación del error estándar y la consecuente inflación del error tipo I (falsos positivos). Incluso métodos diseñados para muestras pequeñas como vcovCR (ajuste tipo CR2), que pueden ser válidos a partir de 10 a 15 clústeres, resultan inestables con tan solo 4 unidades debido a la insuficiencia de grados de libertad efectivos (EDF). Al añadir el departamento directamente en la fórmula como efecto fijo, el modelo controla explícitamente la heterogeneidad geográfica de base, permitiendo comparaciones ajustadas (como Lima vs. otros departamentos) e inferencias válidas sobre la prevalencia de anemia y ferropenia.
